# WeatherApp
This repo will contain 3 separate projects hosted on docker: REST API in Java 21 + Spring boot with Hibernate ORM, Web App in Vue.js (TypeScript) + bootstrap, Mobile App in Flutter (Dart)
## Tech stack:
PostgreSQL <br>
Java 21, Spring, Hibernate <br>
Vue.js, Bootstrap <br>
Flutter <br>
Docker <br>


## Project Setup - Docker compose file

```sh
docker-compose up --build
```
